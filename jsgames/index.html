Tile Switch Game - Basic JavaScript

This is a small example of a game using JavaScript and basic event features of HTML. Style uses Twitter Bootstrap to manage columns and forms.

This small example lacks a structured model, as state and events are encoded into the elements IDs themselves. A better version would use an MV* framework such as Backbone with auto-updating views connected to the DOM object event handlers. A project for next time...!
